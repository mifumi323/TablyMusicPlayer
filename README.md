# Tably Music Player 🎶

**Tably Music Player (Tably MP)** は、ローカルストレージに特化した、開発者自身の満足を徹底的に追求した Windows 向け音楽再生アプリケーションです。

## 🌟 Tably Music Playerの特徴

Tably MP は、一般的なプレイヤーの使いやすさを持ちながら、ニッチなニーズを満たすためのこだわりが凝縮されています。

### 1. 整理と視認性を重視した表形式（Table-View）UI
* 大量の音楽ファイルを効率的に管理できるよう、情報を詳細に表示できる**表形式（Table-View）**のUIを基本としています。
* 「Tably」の名の通り、機能ごとに分離された**タブレイアウト**を採用し、直感的で整理された操作画面を提供します。

### 2. ループ再生への深い対応
* 特にゲーム音楽などの再生を考慮し、**曲ごとに正確なループポイント**（開始・終了）を設定し、無限に再生できる機能を搭載しています。
* 一曲に対して複数の再生設定（音量、EQなど）をプリセットとして持たせ、**バリエーション豊かに楽しむ**ことができます。

### 3. 安定性と快適な操作感
* UI操作と音楽再生処理を**スレッドで完全に分離**しています。UIを操作している最中に音が途切れるといったストレスを完全に排除し、安定した再生環境を提供します。
* （※ 将来的に）オンラインストリーミングのサポートも視野に入れつつ、まずは**ローカルファイル**の再生体験の向上にフォーカスしています。

## 💻 技術情報（Windows向け）

* **動作環境:** Windows OS
* **言語:** C# / Avalonia UI
* **DPI Awareness:** 高解像度ディスプレイでも快適に使用できるDPI対応UI。
* **ライセンス:** 本プロジェクトはオープンソースです。（Apache-2.0 license）

## 🏗️ 開発者の思い

このプロジェクトは、**「個人的に心から満足できる究極のローカル音楽プレイヤー」**を目指して作成されています。万人受けを目指していません。開発者自身のニーズを最優先するため、シンプルな構造と、特定の機能（特にループ再生と表形式UI）への深いこだわりが特徴です。

## 🛠️ ビルドと実行方法

Visual Studio 2022 以降を使用して、リポジトリをクローンし、ソリューションファイルを開いてビルドしてください。必要なNuGetパッケージは自動的に復元されます。

拡張機能の「Avalonia for Visual Studio」をインストールすると、Avalonia UIのデザインビューが利用可能になります。
